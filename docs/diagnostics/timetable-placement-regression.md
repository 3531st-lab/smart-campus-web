3. **学期/周次筛选错误（不是下沉根因）**：筛选会导致课程缺失，但不会生成网格自动安置。
4. **旧缓存资源（排除为根因）**：线上资源与修复前本地哈希一致；本次入口增加 `?v=155` 主动刷新缓存。
